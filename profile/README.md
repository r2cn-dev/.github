## RISC-V Rust for Cloud Native 开源实习计划

R2CN（RISC-V Rust for Cloud Native）开源实习计划由中科院软件所、华为等机构联合主办，目标是培养精通 Rust 编程语言、熟悉 RISC-V 架构的、构建兼容 RISC-V 云原生技术栈创新生态。
